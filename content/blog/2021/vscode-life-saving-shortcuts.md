## Mac 조합키 일람

-   command(또는 cmd) `⌘`
-   shift `⇧`
-   option(또는 alt) `⌥`
-   control(또는 ctrl) `⌃`
-   caps lock `⇪`
-   `fn`

## 자주 쓰는 바로가기 키

VS Code의 모든 바로가기 키 모음은 다음으로 열 수 있다

> `⌘` + `K` + `⌘` + `S`

### 브라우징

-   편집기 탭 이동 `⌥` + `⌘` + `←`/`→`
-   숫자 지정하여 편집기 탭 이동 `⌃` + `숫자`
-   파일로 이동 `⌘` + `P` (`→`를 눌러 여러개의 파일 열기)
-   편집기/에디터 토글 `⌃` + `` ` ``
-   편집기/탐색기 토글 `⌃` + `⇧` + `E`

### 코드 편집

-   주석 토글 `⌘` + `/`
-   자동 줄맞춤 (영역 선택 후) `⌘` + `K` + `F`
-   일치하는 괄호로 이동 `⇧` + `⌘` + `\`
-   줄 선택 `⌘` + `L`
-   현재 행 줄이동 `⌥` + `↑`/`↓`
